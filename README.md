# Bootstrap 5 Visual Guide

A beginner friendly Bootstrap 5 learning resource that explains Bootstrap classes visually.

## About

This project was created to help developers understand how Bootstrap classes work compared to traditional CSS.

Instead of only reading documentation, users can visually see what each Bootstrap class does and learn the CSS concepts behind it.

The goal is simple:

* Learn Bootstrap visually
* Understand the equivalent CSS
* See real examples
* Explore related classes in one place

## Learning Structure

Each topic contains:

### Visual Example

See how the Bootstrap class affects the UI.

### CSS vs Bootstrap

Compare traditional CSS with the Bootstrap equivalent.

### Related Classes

Explore similar Bootstrap classes for the same feature.

### Beginner-Friendly Explanations

Short and simple explanations designed for quick learning.

## Why This Project?

Bootstrap documentation is powerful, but beginners often struggle to understand what classes actually do.

This project bridges that gap by showing:

* Visual demonstrations
* Bootstrap class usage
* Equivalent CSS concepts
* Related utility classes

all in one place.
