border

Border classes add, remove, color, and round borders without writing custom CSS.

Visual:

border
border-primary
border-3
rounded-circle

Difference:

CSS class

.box {

border: 1px solid #dee2e6;

border-radius: 0.375rem;

}

BS class

border

rounded

Border Classes:

BS Class What this does
border Adds border.
border-0 Removes border.
border-top Top border only.
border-bottom Bottom border only.
border-start Left border only.
border-end Right border only.

Border Colors:

  • border-primary
  • border-secondary
  • border-success
  • border-danger
  • border-warning
  • border-info
  • border-light
  • border-dark
  • border-white

Border Width:

  • border-1
  • border-2
  • border-3
  • border-4
  • border-5

Rounded Corners:

BS Class What this does
rounded Default rounded corners.
rounded-0 No rounded corners.
rounded-1 Small rounded corners.
rounded-2 Medium rounded corners.
rounded-3 Large rounded corners.
rounded-4 Extra large rounded corners.
rounded-5 Maximum rounded corners.
rounded-circle Makes a circle.
rounded-pill Makes a pill shape.
