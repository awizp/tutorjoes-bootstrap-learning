text alignment

Text alignment classes control where text appears horizontally.

Visual:

text-start

text-center

text-end

Difference:

CSS class

.text {

text-align: center;

}

BS class

text-center

Related:

BS Class What this does
text-start Aligns text to the left.
text-center Aligns text to the center.
text-end Aligns text to the right.
