1. row

Creates a horizontal row for columns.

CSS class

Same as .row

BS class

row

Row Column Classes

These automatically create columns inside a row.

BS class What this does
row-cols-1 1 column per row.
row-cols-2 2 columns per row.
row-cols-3 3 columns per row.
row-cols-4 4 columns per row.
row-cols-5 5 columns per row.
row-cols-6 6 columns per row.
row-cols-auto Width based on content.

Responsive versions:

  • row-cols-sm-*
  • row-cols-md-*
  • row-cols-lg-*
  • row-cols-xl-*
  • row-cols-xxl-*
