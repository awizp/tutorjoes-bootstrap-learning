Margin and Padding

Margin adds space outside an element. Padding adds space inside an element.

Visual:

Margin (Outside Space)

Parent Container
m-5

Padding (Inside Space)

p-5

Difference:

CSS class

.margin {

margin: 3rem;

}


.padding {

padding: 3rem;

}

BS class

m-5

p-5

Related:

BS Class What this does
m-0 to m-5 Margin all sides.
p-0 to p-5 Padding all sides.
mt-* Top margin.
mb-* Bottom margin.
ms-* Left margin.
me-* Right margin.
mx-* Left & right margin.
my-* Top & bottom margin.
pt-* Top padding.
pb-* Bottom padding.
ps-* Left padding.
pe-* Right padding.
px-* Left & right padding.
py-* Top & bottom padding.
m-auto Auto margin.

Spacing Scale:

  • 0 = 0rem
  • 1 = 0.25rem
  • 2 = 0.5rem
  • 3 = 1rem
  • 4 = 1.5rem
  • 5 = 3rem

Responsive versions:

  • m-sm-* / p-sm-*
  • m-md-* / p-md-*
  • m-lg-* / p-lg-*
  • m-xl-* / p-xl-*
  • m-xxl-* / p-xxl-*
